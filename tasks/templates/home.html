Django App

Basic web project for a CRUD (Create, Read, Update, Delete) task management system using Django. This project will involve creating a web application where users can manage their tasks.

Sign In Sign Up

Key Features

  • User Authentication

    Users must be able to sign up, log in, and log out. Only authenticated users will be able to manage their tasks.

  • Task Listing

    Users should see a list of their tasks on the homepage. Each task should display key information such as title, status, and due date.

  • Task Creation

    Authenticated users can create new tasks by providing a title, description, status, and due date. The default status can be "To Do

  • Task Details

    Clicking on a task should take the user to a detailed view where they can see the full information about the task

  • Task Update

    Users should be able to edit and update the details of a task, including modifying the title, description, status, and due date

  • Task Deletion

    Users should have the option to delete a task.

Technology Stack

  • Django

    Backend framework for handling data models, views, and controllers.

  • Django ORM (Object-Relational Mapping)

    For interacting with the database.

  • HTML/CSS

    Frontend development for creating a user-friendly interface.

  • Bootstrap

    Frontend framework for responsive and attractive design.

  • SQLite

    Database for storing task information.

  • Django Forms

    To handle form creation and validation.
